# Django Todo List
Django를 이용하여 만든 간단한 To-Do List 웹 애플리케이션
할 일을 추가,수정,완료 처리할수 있으며 프론트엔드는 AI도구를 활용하여 구현

📌 프로젝트 소개
이 프로젝트는 Django를 활용한 기본적인 CRUD 기능 학습을 목적으로 제작된 웹 서비스
사용자는 할 일을 등록하고, 수정하거나 완료 상태로 변경하면서 간단하게 일정을 관리

⚙️ 기술 스택
- Backend
Python
Django
SQLite

- Frontend
Claude AI (UI 코드 생성)

✨ 주요 기능
1️⃣ 할 일 추가
새로운 To-Do 항목을 등록할 수 있습니다.

2️⃣ 할 일 수정
기존에 등록된 할 일을 수정할 수 있습니다.

3️⃣ 완료 처리
할 일을 완료 상태로 변경할 수 있습니다.
